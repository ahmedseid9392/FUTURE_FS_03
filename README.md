A **real-world e-commerce solution** built for a local boutique business. This project demonstrates how developers can create value for local businesses by providing a complete online presence with inventory management, secure authentication, and seamless shopping experience.

## 🎯 Project Overview

This platform was built as a real-world internship project with the goal of helping a local boutique:
- Establish a professional online presence
- Increase sales through e-commerce capabilities
- Manage inventory efficiently
- Build customer trust and engagement

## ✨ Features

### Customer Features
- 🏠 **Responsive Landing Page** - Beautiful hero section, featured products, and categories
- 🔍 **Product Browsing** - Filter by category, price range, sizes, and colors
- 🛒 **Shopping Cart** - Persistent cart with quantity management
- 💳 **Checkout Process** - Secure order placement with address management
- 📱 **Mobile-First Design** - Optimized for all devices using Tailwind CSS
- 🔐 **User Authentication** - Secure JWT-based login/registration
- ⭐ **Product Ratings** - Customer reviews and ratings system

### Admin Features
- 📊 **Admin Dashboard** - Complete control over products, orders, and users
- 🖼️ **Image Management** - Cloudinary integration for product image uploads
- 📦 **Order Management** - Track and update order status
- 👥 **User Management** - View and manage customer accounts
- 📈 **Inventory Control** - Real-time stock management

### Technical Features
- 🔒 **JWT Authentication** - Secure token-based authentication
- 🗄️ **MongoDB Atlas** - Cloud database with data persistence
- ☁️ **Cloudinary Integration** - Optimized image storage and delivery
- 🚀 **RESTful API** - Well-structured backend endpoints
- 📱 **Mobile Responsive** - Fully responsive design
- ⚡ **Performance Optimized** - Fast load times and smooth animations

## 🛠️ Tech Stack

### Frontend
- **React 18** with Vite - Fast development and building
- **Tailwind CSS** - Utility-first styling
- **React Router v6** - Client-side routing
- **TanStack Query** - Data fetching and caching
- **Zustand** - State management for cart
- **React Hook Form** - Form handling with Zod validation
- **Framer Motion** - Smooth animations
- **Axios** - HTTP client
- **React Hot Toast** - Beautiful notifications

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB Atlas** - Cloud database
- **Mongoose** - ODM for MongoDB
- **JWT** - Authentication
- **Bcrypt.js** - Password hashing
- **Cloudinary** - Image upload and management
- **Multer** - File handling
